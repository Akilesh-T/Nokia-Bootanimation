# WatchDogs Bootlogo for Nokia 8
## Description
Replaces the Bootanimation.zip from with a WatchDogs Animated Bootlogo.
Simple as that.

## Changelog
1. v1
  * Initial Release for NB1-8.1
    
2. v2
  * moved to Unity template
  * Works on both Oreo and Pie.
